# CWS_BLSTM_CRF
Chinese Word Segmentation using BLSTM-CRF model (a tensorflow version)


### Experimental Results
---
|models            |F1(PKU)|
|:----------------:|:-----:|
|BLSTM-CRF         |94.94% |
|BLSTM-CRF + bigram|95.80% |

Experiments are conducted on the PKU dataset from the second Sighan Bakeoff.
